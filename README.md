# Crescendo
![Crescendo](./Resources/Brand/crescendo-logo-text-white.png)
A performance oriented game engine being written in C++

It is based off Hazel from the Cherno. Once the base systems are finished it will diverge into its own identity

This engine is not designed for production or out-of-house use. The API's are subject to change at any moment! It is simply transparency for the development of this engine
